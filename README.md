# LabSync

**LabSync** is an IoT-based Laboratory Management and Safety System developed as a Capstone project at **Bulacan State University (Sarmiento Campus)**.

### Features
* **IoT Monitoring:** Uses ESP32 for real-time laboratory safety tracking.
* **Management System:** Web-based dashboard for laboratory equipment and security.
* **AI Enhanced:** Developed using Augment Code to ensure efficient and clean logic.
